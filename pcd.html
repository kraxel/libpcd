1. Overview and basic functions

pcd_open
Open a file for reading
pcd_select
Select resolution and area for decoding, set some options. Allocates required memory.
pcd_decode
Does the huffmann decoding (for 4BASE and 16BASE)
pcd_get_image
Transforms the PhotoCD's YUV coding to RGB/GRAY, various output formats are possible.
pcd_get_image_line
dito, but allows reading the image line by line.
pcd_close
Close file.
The pcd_select, pcd_decode, pcd_get_imgdata cycle can run more than once with different parameters without reopening the file.

There are few other functions and useful macros, see the Documentation below for details. You may have a look to the test/example applications too.
